# HR‑аналитика для прогнозирования удовлетворённости и оттока сотрудников»
В рамках учебного проекта разработана система прогнозной аналитики для HR‑отдела. Цель — снизить риски финансовых потерь и оттока персонала за счёт предсказания двух ключевых показателей:

  * уровня удовлетворённости сотрудника работой (в диапазоне 0–1);

  * вероятности увольнения сотрудника.

**Что было сделано:**

  * Проведена полная предобработка данных (загрузка, анализ пропусков, очистка, кодирование признаков).

  * Выполнен исследовательский анализ: изучены взаимосвязи признаков, составлен портрет «уволившегося сотрудника», визуализированы распределения ключевых показателей.

  * Построены пайпланы для получения лучшей модели с подбором гиперпараметров:

    * для уровня удовлетворённости: использованы модели (LinearRegression, SVR, KNeighborsRegressor, DecisionTreeRegressor), оценка по метрике SMAPE ≤ 15;

    * для прогноза увольнения: использованы модели (LogisticRegression, SVC, KNeighborsClassifier, DecisionTreeClassifier), оценка по метрике ROC‑AUC ≥ 0,91.

  * Интегрированы результаты: предсказанный уровень удовлетворённости использован как дополнительный признак для модели увольнения.

  * Подготовлены выводы и рекомендации для бизнеса.
****
**Результат:** созданы работоспособные модели, позволяющие HR‑аналитикам заранее выявлять сотрудников с низким уровнем удовлетворённости и высоким риском увольнения. Это даёт возможность принимать профилактические меры (корректировка нагрузки, пересмотр зарплаты, карьерные диалоги) до возникновения критических ситуаций.

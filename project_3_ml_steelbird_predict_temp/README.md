# Прогноз температуры стали для оптимизации энергозатрат на металлургическом комбинате
**Цель проекта:** разработать модель машинного обучения, которая предсказывает температуру расплавленной стали в процессе обработки. Это позволит снизить потребление электроэнергии и эффективно имитировать технологический процесс на металлургическом комбинате.

**Что было сделано:**

1. **Сбор и консолидация данных.** Объединили 7 разнородных источников данных (о нагреве, подаче материалов, продувке газом, замерах температуры) в единый датафрейм по номеру партии (key).

1. **Предобработка и анализ.** Провели исследовательский анализ каждого набора данных, выявили ключевые закономерности, обработали пропуски и аномалии.

1. **Генерация признаков.** На основе технологического процесса создали новые информативные признаки:

    * полная мощность и потреблённая энергия;

    * длительность нагрева;

    * состав сыпучих и проволочных материалов.

1.  **Обучение моделей.** Протестировали несколько алгоритмов, включая:

    * линейную регрессию;

    * градиентный бустинг (CatBoostRegressor).

1.  **Оптимизация.** Для CatBoost провели подбор гиперпараметров, чтобы добиться максимальной точности.

1. **Оценка качества.** Проверили модели на тестовой выборке по метрике MAE. Требование заказчика: MAE < 6,80.

1.  **Интерпретация.** С помощью SHAP‑анализа определили, какие технологические параметры сильнее всего влияют на температуру.
****
**Результаты:**

* CatBoostRegressor: MAE = 4,2 (на 37 % лучше целевого показателя).

* Линейная регрессия: MAE = 4,6 (также удовлетворяет требованию заказчика).

**Выводы и преимущества:**

*  Модель превосходит требования заказчика по точности прогноза.

*  Две рабочие модели дают гибкость применения:

    *  CatBoost — для максимальной точности при планировании.

    *  Линейная регрессия — для быстрых оперативных решений (высокая скорость и полная интерпретируемость).

**Практические рекомендации:**

1.  Внедрить две модели параллельно:

    *  CatBoost — для точного прогнозирования и планирования.

    *  Линейную регрессию — для оперативного управления и анализа факторов в реальном времени.

1.  Оптимизировать режимы работы электродов и временные циклы нагрева — это ключевые факторы влияния на температуру.

1.  Стандартизировать рецептуры легирования для более предсказуемого теплового баланса.

1.  Интегрировать модель в систему поддержки решений для операторов: вводить плановые параметры и получать прогноз температуры.
****
